# 🔬 GRASP计算配置生成器网页

## 概述

这是一个基于Web的GRASP计算配置文件生成工具，类似于SLURM配置生成器的形式。用户可以通过直观的网页界面快速生成GRASP计算所需的配置文件，无需手动编写复杂的TOML格式配置。

## 功能特点

### 🎯 核心功能
- **可视化配置界面**：直观的表单界面，无需记忆配置格式
- **预设模板**：内置GdI、Ce、Pr等常见原子的配置模板
- **实时预览**：配置修改后立即显示生成的配置文件内容
- **一键下载**：直接下载生成的`config.toml`文件
- **剪贴板复制**：快速复制配置内容到剪贴板

### 📋 配置参数
- **基本信息**：原子符号、组态名称、光谱项
- **路径设置**：根路径、选定CSFs文件名
- **计算参数**：初始选择比例、截断值、计算循环次数、计算方法
- **机器学习参数**：随机森林树数、随机种子、类别权重
- **SLURM设置**：作业名称、分区、任务数、节点数

## 使用方法

### 1. 打开网页
```bash
# 在浏览器中打开HTML文件
open simple_config_generator.html
# 或者双击文件在默认浏览器中打开
```

### 2. 选择预设模板
点击预设按钮快速加载常见配置：
- **GdI原子**：钆离子的标准配置
- **Ce原子**：铈离子的标准配置  
- **Pr原子**：镨离子的标准配置
- **自定义**：手动输入所有参数

### 3. 修改配置参数
根据实际需求调整以下参数：

#### 基本信息
- **原子符号**：如GdI、Ce、Pr等
- **组态名称**：如cv4odd1as3_odd1
- **光谱项**：原子的光谱项描述

#### 路径设置
- **根路径**：计算数据的根目录路径
- **选定CSFs文件名**：已选定的CSFs文件名（不含扩展名）

#### 计算参数
- **初始选择比例**：初始CSFs选择比例 (0.01-1.0)
- **截断值**：混合系数截断值
- **计算循环次数**：最大计算循环次数
- **计算方法**：RCI或RMCDHF

#### 机器学习参数
- **随机森林树数**：随机森林分类器的树数量
- **随机种子**：确保结果可重现的随机种子
- **类别权重**：正负样本的权重设置

#### SLURM设置
- **作业名称**：SLURM作业名称
- **SLURM分区**：计算分区名称
- **每节点任务数**：每个计算节点的任务数
- **节点数量**：使用的计算节点数量

### 4. 生成和下载配置
- 点击"🔄 生成配置"按钮实时更新配置
- 点击"💾 下载配置文件"下载`config.toml`文件
- 点击"📋 复制到剪贴板"复制配置内容

```toml
continue_cal = true
cal_loop_num = 12
cal_error_num = 0
cal_method = "rci"
difference = 0

expansion_ratio = 2
target_pool_file = "cv4odd1as3_odd1.c"
root_path = "/home/workstation3/caldata/GdI/cvodd1/as3_odd1"
selected_csfs_file = "mJ-1-90chosenas3_odd1.c"
selected_csfs_mix_file = "mJ-1-90chosenas3_odd1.cm"

# 收敛性检查参数
std_threshold = 1e-6  # 标准差阈值
relative_std_threshold = 1e-4  # 相对标准差阈值

[model_params]
n_estimators = 1000
random_state = 42

[model_params.class_weight]
0 = 1
1 = 3

# SLURM配置
job_name = "GdIoddImlcias3_odd1"
partition = "work3"
tasks_per_node = 46
node_count = 1

# 生成时间: 2025-01-27T02:30:00.000Z
# 配置生成器版本: 1.0.0
```

## 使用流程

1. **准备阶段**
   - 确定要计算的原子和组态
   - 准备CSFs文件路径
   - 确定SLURM集群配置

2. **配置生成**
   - 打开配置生成器网页
   - 选择或输入原子信息
   - 设置计算参数
   - 配置SLURM参数

3. **文件使用**
   - 下载生成的`config.toml`文件
   - 将文件放置到计算目录
   - 运行GRASP计算脚本

## 注意事项

### ⚠️ 重要提醒
1. **路径验证**：确保输入的路径在目标系统上存在
2. **参数范围**：注意数值参数的合理范围
3. **文件权限**：确保有权限访问指定的目录和文件
4. **集群配置**：确认SLURM分区和节点配置正确

### 🔍 参数说明
- **初始选择比例**：建议范围0.05-0.2，根据CSFs数量调整
- **截断值**：通常设置为1e-09到1e-06之间
- **计算循环次数**：根据收敛情况设置，通常10-20次
- **随机森林树数**：建议500-2000，更多树数可能提高精度但增加计算时间

## 故障排除

### 常见问题

**Q: 网页无法打开？**
A: 确保使用现代浏览器，检查文件路径是否正确

**Q: 下载的文件为空？**
A: 检查浏览器是否阻止了下载，尝试允许下载权限

**Q: 复制到剪贴板失败？**
A: 某些浏览器需要HTTPS环境才能使用剪贴板API，可以手动复制内容

**Q: 配置参数无效？**
A: 检查参数范围，确保数值在合理区间内

### 技术支持
如果遇到问题，可以：
1. 检查浏览器控制台是否有错误信息
2. 确认所有必填字段都已填写
3. 尝试重置表单重新配置
4. 参考现有的配置文件格式
